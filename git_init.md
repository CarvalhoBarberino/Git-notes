# git init

## Útil para
- Iniciar um repositório local

## Explicação

Esse comando inicializa um repositório local na pasta onde for executado.

Ele cria a subpasta ".git", que indica a existência de um repositório. Nessa pasta funciona a parte "debaixo do capô" do repositório.

## Sintáxe/exemplo
```
git init
```
