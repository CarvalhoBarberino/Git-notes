# git clone

## Útil para
- Clonar um repositório remoto.

## Explicação
No linguajar do git, se diz "clonar" ao invés de "baixar" um repositório.

## Sintáxe/exemplo
Apenas clonar.
```
git clone https://github.com/CarvalhoBarberino/projectFive.git
```
Clonar e dar nome a pasta clonada.
```
git clone https://github.com/CarvalhoBarberino/projectFive.git projetoCinco
```
Clonar apenas um branch.
```
git clone https://github.com/CarvalhoBarberino/projectFive.git --branch julioLocal --single-branch
```
